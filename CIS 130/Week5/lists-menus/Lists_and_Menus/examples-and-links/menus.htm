Basic steps for a list menu

  1. Create a basic list
  2. Remove bullets and padding from the list (ul or ol)
  3. Set margin on list (ul or ol)
  4. Set width of list (ul or ol)
  5. Remove the underline from anchors
  6. Maybe add a line to the li or a using the border property
  7. Set background colors - ul and to a
  8. Change the display of the a from inline to block
  9. IE bug: may add margin to bottom of li. To fix: set width of links to 100%
  10. Add some padding to the links; maybe center them?
  11. Adjust link width (if set) (width = width - padding left - padding right)
