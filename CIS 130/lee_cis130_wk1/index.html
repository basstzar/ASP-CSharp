HTML and CSS Review

Cascading Style Sheets (CSS) is the code that allows us to manipulate the look and feel of an HTML based page. Proper use of CSS allows for full styling access to all of the HTML elements while still allowing for proper semantic markup. Here's a site that offers a basic overview and an introductory lesson from another site.

What's CSS?

CSS is for users the one thing that must always be right. An average person browsing the web might never know if your PHP, Rails, etc. is less than elegant but make a mistake in matching colors or image placement and they will instantly know that something is off. CSS is what creates the user interface - it's what keeps users on your page. It's the magic that matters.
